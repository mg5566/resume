## 프로젝트 정보
- **프로젝트명**: resume
- **기술 스택**: React 19 + TypeScript + Vite
- **패키지 매니저**: pnpm (9.5.0+)
- **Node 버전**: >=18.0.0

## 개발 명령어
- `pnpm dev`: 개발 서버 실행
- `pnpm build`: 프로덕션 빌드 (TypeScript 컴파일 + Vite 빌드)
- `pnpm lint`: ESLint 실행
- `pnpm preview`: 빌드된 앱 미리보기

## 개발 도구
- ESLint + Prettier (코드 포맷팅)
- TypeScript 5.8
- Vite 7.0 (번들러)
